# Azure Backup using MARS Agent – File & Folder Recovery Lab
This project demonstrates how to protect on-premises Windows Server data using the Microsoft Azure Recovery Services (MARS) Agent and Azure Recovery Services Vault.

The lab includes:
- MARS Agent installation
- Server registration with Azure
- Backup policy configuration
- File & folder backup
- Retention policy configuration
- File recovery testing
- Server unregistration
