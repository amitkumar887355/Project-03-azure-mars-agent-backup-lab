# Project-03-azure-mars-agent-backup-lab
# Azure Backup using MARS Agent – File & Folder Recovery Lab
This project demonstrates how to protect on-premises Windows Server data using the Microsoft Azure Recovery Services (MARS) Agent and Azure Recovery Services Vault.  The lab includes: - MARS Agent installation, Server registration with Azure, Backup policy, File &amp; folder backup, Retention policy, File recovery testing, Server unregistration
